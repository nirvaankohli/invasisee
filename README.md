# invasisee

InvaSee helps users identify and report invasive species using AI and crowd verification. Discover nearby reports, confirm findings, and earn rewards as your virtual tree grows, turning local action into global environmental protection.

## Landing Page

Visit `index.html` to view the responsive landing page featuring:

- **Hero Section**: "Protect Your Environment." headline with call-to-action buttons
- **How It Works**: 3-step process explanation (Spot & Capture, AI Analysis, Community Verify)
- **Map Preview**: Interactive map showcase for viewing nearby reports
- **Tree Pet Teaser**: Virtual tree growth reward system
- **Mobile-First Design**: Optimized for all screen sizes (mobile, tablet, desktop)

### Design Features

- White background with green accent colors (#2d6a4f palette)
- Subtle nature-inspired animations (floating leaves, growing tree)
- CSS gradients and inline SVG graphics (no heavy images)
- Fully accessible with semantic HTML and ARIA labels
- Responsive design with breakpoints at 768px and 1024px

### Running Locally

Simply open `index.html` in a web browser, or serve it with a local web server:

```bash
python3 -m http.server 8000
```

Then visit `http://localhost:8000` in your browser.
